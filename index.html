Food Bank Analysis - Copenhagen Region

Tools: Python, Clustering, Apriori, Scrapping

This project tackles food waste by creating a data-driven food bank system using surplus food from stores. Data from the Salling Group API, including store locations and items nearing expiration, served as the foundation. The Anti-Food Waste API pro- vided daily updates on surplus items . A constrained agglomerative clustering algorithm grouped stores by proximity, assigning each cluster a food bank location. Recipes scraped from online sources were structured and matched to surplus items for practical utilization. By combining clustering, recipe analysis, and surplus data, the project offers a scalable solution to reduce food waste and support community welfare.

Real vs Fake Discriminator

Tools: CNN, ViT, DIRE, Fine-Tuning

This project focuses on detecting hyper-realistic synthetic images, such as deepfakes, using frequency domain analysis and various machine learning models. By analyzing frequency patterns in a custom dataset of real and synthetic images, three models were evaluated: DIRE, a custom-built CNN, and a fine-tuned vision transformer (ViT). The results demonstrate the potential of combining spectral analysis with advanced machine learning models to enhance synthetic image detection and ensure media authenticity.
